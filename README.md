## Project Title
Side Quest Week 2: Very Angry Blob

---
## Description
This sketch displays a pulsing, angry blob. The user can press a button to make it "pulse" to express its anger. The blob will occasionally pulse by itself. This project explores how shapes can convey emotion and feelings.

---
## Interaction Instructions
Press the on-screen button to make the blob pulse.
